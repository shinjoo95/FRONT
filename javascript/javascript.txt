1. 자바스크립트 코드 작성 방법
- html 파일과 자바스크립트 연결
    - 내부 스크립트 방법 : html 파일에서 script 태그의 콘텐츠 영역에 자바스크립트 코드를 작성하는 방법 
    <body>
        <script>
        document.write("내부 스크립트 방법");
        </script>
    </body>

    - 외부 스크립트 방법 : js 확장자로 된 별도의 파일을 생성하고 생성한 파일에 자바스크립트 코드를 작성한 뒤, html 문서와 연결
    .js
    document.write("외부 스크립트 방법");

    .html
    <body>
        <script src="script.js"></script>
    </body>

- 자바스크립트 코드 실행하기
    - 콘솔창 사용하기 : 자바스크립트는 웹 브라우저에서 실행되는 언어이므로 웹 브라우저의 콘솔창에서 자바스크립트 코드를 바로 작성해 실행할 수 있음

    - 확장 프로그램 사용하기 : VSCode 에서 Code Runner 확장 프로그램을 사용하면 에디터 내부에서 자바스크립트 또는 코드의 실행 결과를 바로 확인 가능 
                            (Code Runner)
